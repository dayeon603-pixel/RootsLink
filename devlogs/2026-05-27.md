## Summary

### `src/serving/app.py` — FastAPI REST API scaffold

Full HTTP interface for real-time EMG-to-speech inference.

**Endpoints:**

| Method | Path | Purpose |
|---|---|---|
| `GET` | `/health` | Liveness probe; `model_loaded` field |
| `GET` | `/config` | Encoder config + window/hop sizes |
| `POST` | `/decode` | Neural window (base64 C×T float32) → articulatory (T_art×12) |
| `POST` | `/adapt` | Fast (norm stats) or deep (placeholder) session adaptation |
| `WS` | `/decode/stream` | Streaming JSON-frame decode |

`_AppState` singleton with `load()` / `reset()` for test isolation.
`load_model()` public entry point for tests + production.

| TestAdapt | 5 | 503 without model; fast/deep 200; invalid strategy 422; round-trip decode |

### `experiments/session_adapt_benchmark.py` — simulation study

Simulates drift (σ ∈ {0.1, 0.3, 0.5, 1.0, 2.0}) and compares:
no-adapt / fast (norm stats) / deep (supervised 50-step fine-tune).
Metric: 1 − cosine_sim(ref_features, drift_features). Outputs Markdown table.
